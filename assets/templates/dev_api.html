TimetableAPI

Lightweight timetable SDK for timetable querying, filtering, analytics, exports, dashboards and integrations.

Designed for schools, colleges, universities, timetable viewers, analytics dashboards and custom timetable rendering systems.

Current Version: 1.0.0

Getting Started

Include your timetable export: 

<script src="timetable.js"></script>

The API will automatically become available globally: 

window.TimetableAPI

Basic Example

const mondaySchedule =
    TimetableAPI.getDaySchedule(1);

console.log(mondaySchedule);

Available Methods

All IDs are strings unless otherwise specified.
getMetadata() METADATA

Returns timetable metadata and configuration. 

const metadata =
    TimetableAPI.getMetadata();
Returns: 
{
    name,
    institution,
    type,
    slot_length,
    days_per_cycle,
    slots_per_day
}
getDay(dayId) LOOKUP

Returns day name from day ID. 

TimetableAPI.getDay(1);

// "Monday"
getSlot(slotId) LOOKUP

Returns slot time range. 

TimetableAPI.getSlot(2);

// "10:00 - 12:00"
getCourse(courseId) LOOKUP

Returns course information. 

TimetableAPI.getCourse("1");
getModule(moduleId) LOOKUP

Returns module information. 

TimetableAPI.getModule("12");
getModuleByCode(moduleCode) LOOKUP

Finds module ID by module code. 

TimetableAPI.getModuleByCode("HCSCI233");
getVenue(venueId) LOOKUP

Returns venue information. 

TimetableAPI.getVenue("2");
getLecturer(lecturerId) LOOKUP

Returns lecturer information. 

TimetableAPI.getLecturer("7");

Schedule Queries

getDaySchedule(day) SCHEDULE

Returns full timetable schedule for a day. 

TimetableAPI.getDaySchedule(1);
getSlotSchedule(day, slot) SCHEDULE

Returns all timetable entries for a slot. 

TimetableAPI.getSlotSchedule(1, 2);
getCourseSchedule(courseId, level = null) SCHEDULE

Returns timetable filtered by course. 

TimetableAPI.getCourseSchedule("1");

TimetableAPI.getCourseSchedule(
    "1",
    "2.1"
);
getLecturerSchedule(lecturerId) SCHEDULE

Returns lecturer timetable. 

TimetableAPI.getLecturerSchedule("4");
getVenueSchedule(venueId) SCHEDULE

Returns venue timetable. 

TimetableAPI.getVenueSchedule("2");

Venue Utilities

getFreeVenues(day, slot) UTILITY

Returns all venues that are free during a specific slot. 

TimetableAPI.getFreeVenues(1, 2);

Course Utilities

getCourseModules(courseId, level = null) UTILITY

Returns modules associated with a course. 

TimetableAPI.getCourseModules("1");

TimetableAPI.getCourseModules(
    "1",
    "2.1"
);

Search

searchModules(query) SEARCH

Search modules by name or module code. 

TimetableAPI.searchModules("security");

TimetableAPI.searchModules("HCSCI");

Formatting

formatEntry(moduleId, entry) FORMATTER

Converts raw timetable entries into human-readable structures. 

TimetableAPI.formatEntry(
    moduleId,
    entry
);

Flat Schedule

getFlatSchedule() ANALYTICS

Returns a flattened schedule structure ideal for:

  • React applications
  • Vue applications
  • Analytics
  • Charts
  • Exports
  • Filtering systems
  • Dashboards
const schedule =
    TimetableAPI.getFlatSchedule();
[
    {
        day: "Monday",
        slot: "08:00 - 10:00",
        module_code: "HCSCI132",
        lecturer: "Mrs N. Sarai",
        venue: "Lab 1"
    }
]

Advanced Usage

data ADVANCED

Direct raw access to timetable data. 

TimetableAPI.data
Raw access is intended for advanced users who need direct timetable structures.

Versioning

TimetableAPI.version
